# Mapa do Sistema

## Componentes principais
- `src/main.asm`: ponto principal para experimentos e patches em Assembly.
- `.vibecoding/context/project_memory.md`: memória acumulada do projeto.
- `.vibecoding/docs/addresses.md`: endereços de RAM e códigos descobertos.
- `rom-base/`: ROM original e arquivos de tradução.
- `referencias/sprites/`: imagens de referência para sprites.

## Relações
- A memória do projeto orienta a pesquisa técnica.
- A documentação de endereços apoia o uso do debugger e do memory viewer.
- As referências visuais ajudam na substituição de sprites e na validação estética.
